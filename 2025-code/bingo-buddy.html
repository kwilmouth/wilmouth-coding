Bingo Buddy

Call out random bingo numbers and keep the game moving with a simple, no-fuss number generator for 75-ball bingo games.

Last: --
 
 

How to Play

  1. Call the Numbers.
    Click the "Call the Next Number" button to reveal a random call from the standard 75-ball bingo set (B1 to O75). Each number is unique and will only be called once per game.
  2. Track Calls.
    As each number is called, it will be highlighted in the master grid so you can easily keep track of what’s already been announced.
  3. Start Fresh
    Click “New Game” to reset the board and begin a brand new round when you're ready to start again.
