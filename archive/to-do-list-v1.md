# To-Do List

## Problem

As an E$ user, I need to track to-do items so that I may manage the steps I need to take to align my financial situation with the Ramsey Plan.

## User Stories

- As a user, I need to view my list of to-do items so that I may decide what next action to take.
- As a user, I need to add a to-do item to the list so that I may keep the list up-to-date.
- As a user, I need to mark a to-do item complete on the list so that I may track my progress.
- As a user, I need to remove a to-do item from the list so that I may keep the list up-to-date.
